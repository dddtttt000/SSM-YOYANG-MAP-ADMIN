## 🛠️ 8. 기술적 요구사항

### 🔧 개발 환경
- **Frontend**: React 18 + TypeScript + Vite
- **UI Library**: Chakra UI v2.8.2
- **State Management**: TanStack Query v5
- **Backend**: Supabase (PostgreSQL + Auth + RLS)
- **권한 관리**: Row Level Security (RLS) 정책
